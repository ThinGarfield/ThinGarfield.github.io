Currently supported operations:
  • normal +, -, *, /   :   following common sense operator precedence.
  • x^y   :   $x^y$
  • x!   :   the factorial of $x$
  • x!!   :   means the double-factorial of $x$
  • \binom{n}{k}   :   $\binom{n}{k}$
  • \frac{a}{b}   :   $\frac{a}{b}$
  • a \times b, a \cdot b, a * b  :   all means multiplication
  • gcd(a,b)   :   the greatest common divisor. Supports arbitrary number of arguments.
  • lcm(a,b)   :   the least common multiple. Supports arbitrary number of arguments.
  • isprime(n)  :   whether $n$ is a prime number
  • nextprime(n)  :   the smallest prime number greater than $n$
  • a mod b or a \pmod{b} :   $a \pmod{b}$. Note that
    • $b$ must be positive integer.
    • the mod operator has lower precendence than arithmetic operators, x + y mod m means $(x+y)\pmod{m}$.
    • a / b mod m means $a \cdot b^{-1} \pmod{m}$
